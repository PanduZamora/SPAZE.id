== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Important: You need to have installed and activated ShareMe
2. Install the plugin via WordPress or download and upload the plugin to the /wp-content/plugins/
3. Activate the plugin through the 'Plugins' menu in WordPress
4. You can see the 'ShareMe' Share option on your every blog posts
